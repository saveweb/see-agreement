    Microsoft 隱私權聲明

标题: Microsoft 隱私權聲明
时间版本: 2021年12月

    简介#

    我們很重視您的隱私權。 本隱私權聲明旨在說明 Microsoft 處理的個人資料、Microsoft 處理方式及其用途。

    Microsoft 提供範圍廣泛的產品，包括全球企業經營不可或缺的伺服器產品、住家使用的裝置、學生校內使用的軟體，以及協助開發人員研發創新或提供新興主機服務的各項服務。 本聲明內所稱的 Microsoft 產品，包括了 Microsoft 服務、網站、應用程式、軟體、伺服器以及裝置。

    請詳閱本隱私權聲明的特定產品詳細資料，了解更多相關資訊。 本聲明適用於 Microsoft 與您進行的互動，以及下列 Microsoft 產品，乃至於顯示本聲明內容的其他 Microsoft 產品。

    青少年可能較喜歡以青少年的隱私權頁面做為開始使用。。 該頁面會強調可能對青少年有助益的資訊。

    正文#

    我們收集的個人資料#

    基於以下所述各種目的 (包括為了要有效運作，以及提供您使用我們產品的最佳體驗)，Microsoft 透過與您的互動以及本公司產品，向您收集資料。 當您在某些情況下，例如在您建立 Microsoft 帳戶、管理您組織的授權帳戶、向 Bing 提交搜尋查詢、註冊 Microsoft 活動、向 Cortana 發出語音命令、上傳文件至 OneDrive、註冊 Microsoft 365，或向我們申請支援時，會直接提供部分的該資料。 有些則是透過收集您與本公司產品的互動、使用及體驗以及雙方通訊的相關資料來取得。

    我們基於各種法律原因及許可 (有時候稱為「法律基礎」) 及以下所述各種目的來處理資料，包括經您同意後，合法權益的平衡、簽訂及履行合約之需要，以及遵守法律義務。

    我們也會從第三方取得資料。 我們依據本聲明中所述方式保護從第三方取得的資料，外加資料來源所套用的其他任何限制。 這些第三方來源會隨時間產生變化，包括︰

    • 資料仲介商，我們向其購買人口統計資料來補充所收集資料的不足。
    • 服務，將其服務使用者產生的內容提供予其他人，例如當地商家評論或公開社交媒體文章。
    • 通訊服務，包括電子郵件提供者和社交網路，當您允許我們存取您在這類第三方服務或網路上的資料時。
    • 服務提供者，以協助我們判斷您的裝置位置。
    • 合作夥伴，我們與其合作提供共同品牌服務，或參與聯合行銷活動。
    • 開發人員，透過 Microsoft 產品建立體驗或建立 Microsoft 產品體驗。
    • 透過 Microsoft 產品提供體驗的第三方。 公開可用的來源，例如，開放式公部門、學術及商業資料集及其他資料來源。

    如果您代表運用 Microsoft 企業及開發人員產品的組織 (例如企業或學校)，請參閱本隱私權聲明的企業及開發人員產品一節，了解我們如何處理您的資料。 如果您是 Microsoft 產品的終端使用者或擁有公司所提供的 Microsoft 帳戶，請參閱您的組織所提供的產品和Microsoft 帳戶章節以取得詳細資訊。

    當涉及您使用的技術和分享的資料時，您有選擇的權利。 在您被要求提供個人資料時，您可以拒絕。 我們的許多產品都需要一些個人資料來操作及為您提供服務。 如果您選擇不提供我們在操作和提供產品或功能時所需的資料，便無法使用該產品或功能。 同樣地，當我們依法需要收集個人資料，以便與您簽訂合約或履行合約，而您未提供資料，我們將無法簽訂合約；或如果這與您正在使用的現有產品有關，我們可能會有暫止或取消該產品。 如果是這種情形時，我們會通知您。 如果提供資料是選擇性的，而您選擇不分享個人資料，使用此資料的功能 (如個人化) 將不適用於您。

    我們所收集的資料，取決於您與 Microsoft 的互動內容，以及您的選擇 (包括您的隱私權設定)、您所使用的產品及功能、位置和相關法律.

    我們收集的資料包括下列項目：

    姓名和連絡人資料. 您的姓名、電子郵件地址、郵寄地址、電話號碼以及其他類似的連絡人資料。

    認證. 密碼、密碼提示以及用於驗證和帳戶存取的類似安全資訊。

    人口統計資料. 關於您的資料，例如您的年齡、性別、國家/地區及慣用語言。

    付款資料. 處理付款所需的資料，例如您的付款工具號碼 (如信用卡號碼)，以及與您的付款工具相關聯的安全代碼。

    訂閱和授權資料. 您的訂閱、授權及其他權利相關資訊。

    互動. 關於您使用 Microsoft 產品的資料。 在某些情況下 (例如搜尋查詢)，這是您為了使用產品所提供的資料。 在某些情況下 (例如錯誤報告)，這是我們產生的資料。 互動資料的其他範例包括：

    • 裝置與使用狀況資料. 您的裝置以及您使用的產品和功能的相關資料，包括硬體和軟體、產品效能，以及您的設定等資訊。 例如：
      • 付款和帳戶歷程記錄。有關您所購買項目以及與您帳戶相關聯活動的資料。
      • 瀏覽歷程記錄。您所瀏覽網頁的相關資料。
      • 裝置、連線性，以及設定資料。關於您的裝置、裝置設定，以及附近網路的資料。例如，與裝置上安裝的作業系統與其他軟體有關的資料，包括產品金鑰。再加上，IP 位址、裝置識別碼 (例如手機的 IMEI 編號)、區域和語言設定，以及裝置附近的 WLAN 存取點的相關資訊。

      • Bot 使用方式資料。和第三方 Bot 的互動以及和 Microsoft 產品所提供的技能。
    • 興趣與愛好. 您的興趣與「我的最愛」相關資料，例如追蹤關注的運動團隊、偏好的程式語言、追蹤的個股，或加入追蹤天氣或交通等項目的城市。 除了您明確提供的資料之外，我們也有可能根據我們所收集的其他資料，來推斷或導出您的興趣和愛好。
    • 內容使用資料. 您透過本公司產品存取的媒體內容的資訊 (例如電視、影片、音樂、音訊、文字書籍、應用程式和遊戲)。
    • 搜尋與命令. 當您使用具有搜尋或相關生產力功能的 Microsoft 產品時，搜尋查詢及命令。
    • 語音資料. 您的語音資料 (有時候稱為「語音剪輯」)，例如搜尋查詢、命令或您說話的聽寫，可能會包含背景音效。
    • 簡訊、筆跡和輸入資料. 簡訊、筆跡、輸入資料和相關資訊。 例如，當我們收集筆跡資料時，我們收集關於筆跡工具在裝置上的位置資訊。
    • 影像. 影像和相關資訊，例如圖片中繼資料。 例如，當您使用 Bing 影像搜尋服務時，我們會收集您所提供的影像。
    • 聯絡人與關係. 若您使用產品與其他人分享資訊、管理連絡人、與其他人通訊或改善生產力，則本公司會收集您的連絡人與相互關係資料。

    Cookie 是置於裝置上的小型文字檔案，以供發出 Cookie 的網域中網頁伺服器取回所儲存的資料。 此資料通常包含一串能唯一識別您電腦的數字和字母，但是也可能包含其他資訊。 有些 Cookie 是由第三方代表我們放置的。 我們使用 Cookie 及類似技術以儲存及採用您的喜好設定和設定、使您能夠登入、提供以興趣為導向的廣告、打擊詐欺行為、分析產品效能，以及履行以下所述的其他合法用途。 Microsoft 應用程式使用了額外的識別碼 (例如 Windows 中的廣告識別碼) 來達成類似目的，我們許多網站及應用程式也包含 Web 指標或其他類似技術，如下所述。

    廣告識別碼設定不適用於其他由 Microsoft 或協力廠商所提供的興趣導向廣告方法，例如網站上用來提供興趣導向展示型廣告的 Cookie。 在 Windows 上存取或安裝的協力廠商產品可能也會遵循其本身的隱私權原則規範，提供其他形式的興趣導向廣告。 Microsoft 同時採用直接的方式，以及與協力廠商廣告提供者合作的方式，在特定 Microsoft 產品中提供其他形式的興趣導向廣告。 如需有關 Microsoft 如何將資料用於廣告的詳細資訊，請參閱本聲明的 我們如何使用個人資料 一節。

    診斷#

    • 裝置、連線能力與設定資料︰

      • 作業系統及與其設定相關的資料，例如作業系統版本和組建編號、地區和語言設定、診斷資料設定，以及裝置是否屬於 Windows 測試人員計畫的一部分。
      • 已連線的周邊設備相關資料，例如型號、製造商、驅動程式與相容性資料。
      • 與裝置上安裝的應用程式相關資料，例如應用程式名稱、版本與發行者。
    • 裝置是否已就緒進行更新，以及是否有因素可能會妨礙接收更新的功能，例如電力偏低，磁碟空間有限或連線透過付費網路取得。
    • 更新是否成功完成，或是失敗。
    • 診斷收集系統本身可靠性的相關資料。
    • 基本錯誤報告，即您裝置上所執行之作業系統及應用程式的相關健康情況資料。 例如，基本錯誤報告可讓我們了解應用程式 (例如 Microsoft 小畫家或協力廠商遊戲) 是否停止回應或是當機。

    選擇性診斷資料 包含有關裝置、其設定、功能及裝置健康情況的詳細資訊。 選擇性診斷資料也包含您瀏覽的網站、裝置活動 (有時稱為使用方式)，以及可協助 Microsoft 為所有使用者修正並改善產品及服務的增強型錯誤報告等相關資料。 當您選擇傳送選擇性診斷資料時，除了始終包含在內的必要診斷資料，我們還會收集以下附加資訊：

    • 除了依據必要診斷資料所收集的資料之外，有關裝置、連線能力及設定的其他資料。
    • 除了依據必要診斷資料所收集的更新及診斷系統相關資料之外，作業系統和其他系統元件的健康情況相關狀態及記錄資訊。
    • 應用程式活動，例如裝置上啟動哪些程式、其執行時間長度，以及其對輸入的回應速度。
    • 瀏覽器活動，包括 Microsoft 瀏覽器 (Microsoft Edge 或 Internet Explorer) 的瀏覽歷程記錄和搜尋字詞。
    • 增強的錯誤報告，包括發生系統或應用程式損毀時裝置的記憶體狀態 (這可能會在問題發生時，無意間包含使用者內容，如您當時使用之檔案的其中部分)。 損毀資料從未用於量身打造的體驗，如下所述。

    意見反應中樞#

    意見反應中樞是預先安裝的應用程式，其提供一種方式來收集有關 Microsoft 產品和已安裝之第一方和協力廠商應用程式的意見反應。 您可以使用您個人 Microsoft 帳戶，或您用於登入 Microsoft 產品的組織 (例如您的雇主或學校) 所提供的帳戶，登入意見反應中樞。 使用您的工作或學校帳戶登入，可讓您將與您組織關聯的意見反應提交給 Microsoft。

    任何您提供的意見反應，無論您使用的是公司或學校帳戶或個人 Microsoft 帳戶，根據貴組織系統管理員的設定，皆可能可供公開查閱。 此外，如果您使用工作或學校帳戶提供意見反應，貴組織的 IT 系統管理員即可透過意見反應中樞來檢視您的意見反應。

    當您向 Microsoft 提交問題的意見反應或對某問題新增更多細節時，診斷資料將會被傳送至 Microsoft 以改善 Microsoft 產品和服務。 根據您在 Windows 設定應用程式 [診斷資料和意見反應] 一節的診斷資料設定，意見反應中樞可能會自動傳送診斷資料，或在提供意見反應時選擇傳送給 Microsoft。 根據提交意見反應時所選擇的類別，Microsoft 可能會收集額外的個人資料，以進一步排查問題；例如，當您在混合實境中提交意見反應時，所提交的關于位置服務的意見反應或注視相關資訊的位置相關資訊。 當您提交您的意見反應時，Microsoft 也可能會與 Microsoft 合作夥伴 (例如裝置製造商或韌體開發人員)分享您的意見反應與診斷資料，以幫助他們針對 Windows 及其他 Microsoft 產品和服務使用的產品和服務進行疑難排解。 深入了解 Windows 中的診斷資料.

    定位服務與錄製#

    Windows 定位服務. Microsoft 提供位置服務以協助判斷特定 Windows 裝置所在的準確地理位置。 依據裝置的功能，判斷出裝置位置會有不同程度的準確性，而在某些情況下卻可以精確無比。 您已在 Windows 裝置上使用位置服務，或者您擁有 Microsoft 應用程式之特定權限來存取非 Windows 裝置上的位置資訊時，Microsoft 會收集與基地台以及 Wi-Fi 存取點及其位置相關的資料，並在移除識別收集來源之人員或裝置的任何資料後，加入到位置資料庫中。 Microsoft 會使用這份已解除識別性的位置資訊來改善其定位服務，並在某些情況下，與我們的定位服務提供者合作夥伴 (目前為 HERE，請參閱 https://www.here.com/ 和 Skyhood，請參閱 https://www.skyhook.com) 共享，以該提供者的定位服務。

    如果您的設定允許，Windows 服務和功能、Windows 上執行的應用程式，以及 Windows 瀏覽器中開啟的網站就能透過 Windows 存取裝置的位置。 在您首次安裝 Windows 時，部分功能和應用程式會要求允許使用位置服務，某些會在您首次使用該應用程式時詢問，其他則會在您每次存取裝置位置時詢問。 如需使用裝置位置的特定 Windows 應用程式的詳細資訊，請參閱如本隱私權聲明的Windows 應用程式一節。

    當某應用程式或功能存取裝置的位置，而且您使用 Microsoft 帳戶登入時，您的 Windows 裝置也會將它的位置上傳至雲端，而只要是使用您的 Microsoft 帳戶，且您已授與權限的其他應用程式或服務，都能跨裝置使用該雲端上的資訊。 我們將只保留最後已知位置 (每個新位置都會取代上一個位置)。 關於 Windows 裝置近期位置歷程記錄的資料也會儲存在裝置上，即使未使用 Microsoft 帳戶也一樣，而且部分應用程式和 Windows 功能將可以存取此位置歷程記錄。 您隨時可以在 Windows 的設定應用程式中清除裝置的位置歷程記錄。

    在 Windows 設定應用程式中，您也可以檢視哪些應用程式可以存取裝置的精確位置或您裝置的位置歷程記錄、針對特定應用程式關閉或開啟對裝置位置的存取，或關閉對裝置位置的存取。 您也可以設定預設位置，當位置服務無法偵測出您裝置所在的確切位置時將會使用此預設位置。

    即使在您已關閉對裝置位置的存取權，有些協力廠商的傳統型應用程式和服務仍然會使用其他技術 (例如藍牙、Wi-Fi、行動數據機等)，判斷裝置的精確位置。 深入了解協力廠商傳統型應用程式，以及其如何在裝置的位置設定關閉情況下仍可判斷裝置的位置.

    此外，為了在緊急狀況下加速取得協助，每當您撥打緊急電話時，不論您的位置設定為何，Windows 都會嘗試判斷並分享您的準確位置。 如果您的裝置具有 SIM 卡或是正在使用行動數據服務，電信業者便能存取裝置的位置。 深入了解 Windows 中的位置.

    一般位置. 如果您開啟「位置」服務，無法使用您準確位置的應用程式可能會存取您的一般位置，例如您所在的城市、郵遞區號或地區。

    尋找我的裝置. [尋找我的裝置] 功能可讓 Windows 裝置的系統管理員從 account.microsoft.com/devices 尋找該裝置的位置。 若要啟用 [尋找我的裝置]，系統管理員需要使用 Microsoft 帳戶登入，並已啟用位置設定。 即使其他使用者已拒絕所有其應用程式存取位置，此功能還是能發揮作用。 當系統管理員嘗試找出裝置的位置時，使用者將會在通知區域看到通知。 深入了解 Windows 中的 [尋找我的裝置].

    錄製. 部分 Windows 裝置具有錄製功能，可讓您在裝置上擷取您活動的音訊與視訊短片，包括您與其他人的通訊。 如果您選擇錄製一段內容，該錄製將會本機儲存在裝置上。 有時候，您可能會選擇將錄製內容傳送到會公開廣播錄製內容的 Microsoft 產品或服務。 重要事項：在錄製及/或傳送任何通訊之前，您應先了解您的法律責任。這包括事先取得所有參與交談的人員或任何其他授權單位的同意。Microsoft 對於您如何使用錄製內容或錄製功能不負任何責任。

    安全性與安全功能#

    裝置加密. 裝置加密可透過 BitLocker 磁碟機加密技術，對儲存在您裝置上的資料加密，從而保護它們。 當開啟裝置加密時，Windows 會自動加密安裝 Windows 的磁碟，並產生一個修復金鑰。 您個人裝置的 BitLocker 修復金鑰會自動線上備份至您個人的 Microsoft OneDrive 帳戶中。 Microsoft 不會將您的個人修復金鑰用於任何用途。

    惡意軟體移除工具. 做為 Windows Update 的一部分，惡意軟體移除工具 (MSRT) 每個月至少會在您的裝置上執行一次。 MSRT 會檢查裝置是否受到特定及常見的惡意軟體 (「惡意程式」) 感染，並協助移除任何被感染的內容。 MSRT 執行時，將移除您裝置上存在之 Microsoft 支援網站上所列的惡意程式碼。 在惡意程式檢查過程中，將向 Microsoft 傳送一份載有關於所偵測出的惡意程式、錯誤和其他關於您裝置的資料的報告。 如果您不希望 MSRT 將此資料傳送至 Microsoft，您可以停用 MSRT 的報告元件。

    Microsoft 家庭. 家長可使用 Microsoft 家庭來了解其子女使用裝置的情況，並設定限制。 有很多功能可供家庭成員使用，因此當您建立或加入家庭時，請務必仔細檢閱所提供的資訊。 如果您居住在需要權限才能建立帳戶以存取 Microsoft 服務的地區，可能會收到要求或給予家長同意的提示。 如果使用者在地區中未滿法定年齡，在註冊程式期間，系統會提示使用者輸入成人的電子郵件，要求家長或監護人之同意。 對兒童啟用家庭活動報告後，Microsoft 將收集關於兒童如何使用其裝置的詳細資料，並向家長提供其子女的活動報告。 網路動向報告經存留短暫期間後，將依例行程序從 Microsoft 伺服器刪除。

    Microsoft Defender SmartScreen. Microsoft Defender SmartScreen 會在您使用我們的服務時，檢查下載檔案和 Web 內容是否存在惡意的軟體、可能不安全的網路內容，以及其他對您或您裝置的威脅，以協助保護您。 檢查檔案時，該檔案的相關資料會被傳送到 Microsoft，包括檔案名稱、檔案內容雜湊、下載位置，以及檔案的數位憑證。 如果 Microsoft Defender SmartScreen 辨識出檔案為未知或可能不安全的檔案，您就會在開啟檔案之前收到警告。 檢查 Web 內容時，系統會將有關該內容以及您的裝置的資料傳送至 Microsoft，包括內容的完整網址。 如果 Microsoft Defender SmartScreen 偵測出內容可能不安全，您看到的會是警告，而不是該內容。 Microsoft Defender SmartScreen 可在 [設定] 中加以開啟或關閉。

    Microsoft Defender 防毒軟體. Microsoft Defender 防毒軟體會在您的裝置上尋找惡意程式碼和別的垃圾軟體、可能不需要的應用程式，以及其他惡意的內容。 如果沒有任何其他反惡意程式碼軟體在主動保護裝置，系統會自動開啟 Microsoft Defender 防毒軟體來協助保護您的裝置。 如果 Microsoft Defender 防毒軟體已開啟，此軟體便會監視您裝置的安全性狀態。 當 Microsoft Defender 防毒軟體已開啟，或因為啟用 [有限的定期掃描] 而正在執行時，此軟體會自動傳送報告給 Microsoft (報告包含可疑惡意程式碼和別的垃圾軟體、可能不需要的應用程式以及其他惡意的內容等相關資料)，或許還會傳送可能包含惡意內容 (例如惡意程式碼或不明檔案) 的檔案，以便做進一步檢查。 如果報告可能含有個人資料，系統就不會自動傳送報告，而會在傳送前提示您。 您可以將 Microsoft Defender 防毒軟體設定成不要傳送報告和可疑惡意程式碼給 Microsoft。

    語音、語音啟用、筆跡與輸入#

    語音. Microsoft 不但提供以裝置為基礎的語音辨識功能，也提供雲端式 (線上) 語音辨識技術。

    開啟線上語音辨識設定可讓應用程式使用 Microsoft 雲端式語音辨識。 此外，在 Windows 10 中，線上語音辨識設定可讓您在 Windows 內使用聽寫功能。

    設定 HoloLens 裝置或安裝 Windows Mixed Reality 時開啟語音功能，可讓您使用語音命令、聽寫功能以及和應用程式互動。 以裝置為基礎的語音辨識和線上語音辨識設定都會啟用。 在同時啟用這兩種設定的情況下，若開啟裝置的耳機，耳機就一律會聆聽您的語音輸入，並將您的語音資料傳送至 Microsoft 的雲端式語音辨識技術。

    當您使用 Microsoft 的雲端式語音辨識技術時，無論是透過線上語音辨識設定啟用，或當您與 HoloLens 或語音輸入互動時，Microsoft 會收集並使用您的語音錄製，在語音資料中建立語音文字的文字轉錄，以提供語音辨識服務。 未經您的許可，Microsoft 將不會聆聽您的語音錄音。 如需深入了解 Microsoft 如何管理您的語音資料，請參閱 語音辨識技術。

    您可以使用裝置型語音辨識，這樣就不會將語音資料傳送給 Microsoft。 不過，Microsoft 雲端式語音辨識服務提供的辨識功能會比以裝置為基礎的語音辨識更準確。 關閉 [線上語音辨識] 設定後，不仰賴雲端且只用以裝置為基礎之辨識的語音服務 (例如，朗讀程式和 Windows 語音辨識應用程式) 仍將繼續運作，但是 Microsoft 無法收集任何語音資料。

    您可以隨時關閉線上語音辨識功能。 這將會阻止依賴線上語音辨識設定的任何應用程式將語音資料傳送給 Microsoft。 如果您使用 HoloLens 或 Windows 混合實境頭戴式裝置，則也可以隨時關閉以裝置為基礎的語音辨識。 這樣裝置就不會聆聽您的語音輸入。 深入了解 Windows 中的線上語音辨識 .

    語音啟動. Windows 提供支援的應用程式，能夠依據該應用程式特定的語音關鍵字來回應及採取行動，例如在您說「Cortana」時，可讓 Cortana 聆聽及回應。

    如果您給予應用程式權限來聆聽語音關鍵字，Windows 就會主動聆聽麥克風是否傳來這些關鍵字。 一旦辨識了關鍵字，應用程式就會存取您的語音錄音、可以處理錄音、採取行動並做出回應，例如以語音回答。 應用程式可能會將語音錄音傳送到它自己在雲端的服務以處理命令。 每個應用程式在存取語音錄音之前應該都會先向您要求權限。

    此外，當裝置鎖定時，還是可以啟用語音啟動。 若已啟用，在裝置鎖定時相關應用程式還是會繼續聆聽麥克風是否傳來語音關鍵字，並且可以為任何在裝置附近講話的人啟動應用程式。 在裝置鎖定時，應用程式可以存取同樣的功能與資訊，就和裝置未鎖定時一樣。

    您可以隨時關閉語音啟動。 深入了解 Windows 中的語音啟動 .

    即使您已關閉語音啟動，一些第三方傳統型應用程式與服務仍然會聆聽麥克風，並收集您的語音輸入。 深入了解第三方傳統型應用程式，以及應用程式如何在這些設定已關閉的情況下仍然能夠存取麥克風。 .

    語音輸入. 在 Windows 11 中，聽寫功能已更新並重新命名為語音輸入。 語音輸入就像聽寫一樣，使用線上語音辨識技術來提供語音到文字轉錄服務。 您也可以選擇提供語音剪輯，協助改善語音輸入。 如果您選擇不提供語音剪輯，您仍然可以使用語音輸入。 您隨時可以在語音輸入設定中變更您的選擇。 未經您的許可，Microsoft 將不會聆聽您的語音錄音。 深入了解有關改善 Windows 中的語音輸入。 .

    筆跡與輸入個人化. 您輸入和手寫的詞彙會被收集來向您提供：個人字典、提高字元識別以協助您在裝置上打字和書寫，以及在您打字或書寫時提供文字建議。

    您隨時可以關閉筆跡和輸入個人化功能。 這將會刪除儲存在您的裝置上的資料，例如個人字典。 深入了解有關 Windows 中的筆跡和輸入。.

    更新服務#

    Windows 更新服務包括 Windows Update 和 Microsoft Update。 Windows Update 是為您提供 Windows 軟體和其他支援軟體之軟體更新的服務，例如由裝置製造廠商提供的驅動程式和韌體。 Microsoft Update 是一項可為您提供其他 Microsoft 軟體 (例如 Microsoft 365) 之軟體更新的服務。

    網頁瀏覽器：舊版 Microsoft Edge 和 Internet Explorer#

    本節適用於舊版 Microsoft Edge (版本 44 及以下)。 如需非舊版 Microsoft Edge 的相關資訊，請參閱隱私權聲明的Microsoft Edge一節。

    Microsoft Edge 是 Windows 的預設網頁瀏覽器。 Microsoft 提供的傳統瀏覽器 Internet Explorer 也可以在 Windows 中使用。 當您使用網頁瀏覽器存取網際網路時，您的裝置相關資料 (「標準裝置資料」) 會傳送至您造訪的網站和您使用的線上服務。 標準裝置資料包括您裝置的 IP 位址、瀏覽器類型和語言、存取時間，以及參考網站位址。 上述網站的網頁伺服器可能會記錄這些資料。 被記錄的資料內容和使用方式視乎您造訪的網站和您所使用的 Web 服務的隱私作法。 此外，Microsoft Edge 會將唯一的瀏覽器 ID 傳送到特定的網站，讓我們能發展彙總資料，以用來改善瀏覽器功能和服務。

    此外，關於您如何使用瀏覽器的資料 (例如您的瀏覽歷程記錄、網頁表單資料、Temporary Internet Files 和 Cookie) 也會儲存在您的裝置上。 您可以使用 [刪除瀏覽歷程記錄]，從您的裝置中刪除這些資料。

    Microsoft Edge 可讓您擷取並儲存裝置上的內容，例如：

    • 網頁筆記. 允許您在瀏覽的網頁上建立筆跡標註和文字附註，並裁剪、儲存或共用這些內容。
    • Active Reading. 允許您建立和管理閱讀清單，包括網站或文件。
    • 中心. 允許您輕鬆地在同一個區域內輕鬆管理您的閱讀清單、我的最愛、下載項目和歷程記錄。
    • 網站釘選到工作列. 可讓您釘選最愛網站至 Windows 工作列。 網站將能看到您釘選了哪些網頁，以便能提供通知徽章，讓您知道您看可以到他們的網站查看新內容。

    某些儲存於您裝置上的 Microsoft 瀏覽器資訊將在您使用 Microsoft 帳戶登入時跨其他裝置同步。 例如在 Internet Explorer 中，此資訊將包括您的瀏覽器歷程記錄和我的最愛；而在 Microsoft Edge 中，則包括我的最愛、閱讀清單、自動填寫的表單項目 (例如您的姓名、地址和電話號碼)，而且包含已安裝的擴充功能的資料。 例如，如果您在裝置之間同步 Microsoft Edge 閱讀清單，您選擇儲存至閱讀清單的內容複本將會傳送至每個同步的裝置，以便稍後檢視。 您可以移至 [開始] > [設定] > [帳戶] > [同步您的設定]，以停用 Internet Explorer 的同步功能 (如需詳細資訊，請參閱本隱私權聲明的同步設定一節)。 您也可以透過在 Microsoft Edge 的 [設定] 中關閉同步選項，來停用 Microsoft Edge 瀏覽器資訊的同步功能。

    Microsoft Edge 和 Internet Explorer 使用您的搜尋查詢和流覽歷程記錄，向您提供更快的流覽速度和更適切的搜尋結果。 這些功能包括：

    • 搜尋建議 在 Internet Explorer 中自動將您於瀏覽器網址列輸入的資訊傳送給您的預設搜尋提供者 (例如 Bing)，以便在您每鍵入一個字元時提供搜尋建議。
    • 搜尋及網站建議在 Microsoft Edge 中自動將您於瀏覽器網址列輸入的資訊傳送給 Bing (即使您已選取其他預設搜尋提供者)，以便在您每鍵入一個字元時提供搜尋建議。

    您可以隨時關閉這些功能。 為了提供搜尋結果，Microsoft Edge 和 Internet Explorer 會將您的搜尋查詢、標準裝置資訊及位置 (如果您已啟用位置服務) 傳送給您預設的搜尋提供者。 如果 Bing 是您預設的搜尋提供者，我們將根據本隱私權聲明的 Bing 小節所述方式使用此資料。

    Cortana 可以協助您使用像是「詢問 Cortana」這樣的功能在 Microsoft Edge 中瀏覽網頁。 您隨時可以在 Microsoft Edge 設定中停用 Microsoft Edge 中的 Cortana 協助功能。 如需深入了解 Cortana 如何使用資料以及如何加以控制，請參閱本隱私權聲明的Cortana一節。

    Windows 應用程式#

    Windows 已隨附一些 Microsoft 應用程式，而其他應用程式則可從 Microsoft Store 取得。 其中一些應用程式包括：

    地圖應用程式. [地圖] App 提供以位置為基礎的服務，並使用 Bing 服務處理 [地圖] App 中的搜尋。 當 [地圖] 應用程式可以存取您的位置，而您也在 Windows 中啟用了以位置為基礎的服務時，如果您在 Windows 應用程式支援「@」按鍵功能的文字方塊中使用該按鍵起始搜尋，Bing 服務就會收集您在「@」按鍵之後輸入的文字來提供以位置為基礎的建議。 如需深入了解這些由 Bing 所提供的體驗，請參閱如本隱私權聲明的Bing一節。 如果 [地圖] 應用程式擁有存取您位置的權限，即便該應用程式不在使用中，Microsoft 仍可能從您的裝置收集已去除身分識別資訊的位置資料，用以改進 Microsoft 服務。 您可以透過關閉位置服務或關閉地圖服務應用程式對位置服務的存取，來停用地圖服務應用程式對您位置的存取。

    您可以在 [地圖] 應用程式中追蹤您最愛的地點和近期的地圖搜尋。 您的常用位置和搜尋歷程記錄將被包含在搜尋建議中。 如果您使用 Microsoft 帳戶登入，您的最愛位置、搜尋歷程記錄及某些應用程式設定，將會同步至其他裝置和服務 (例如 Cortana)。 如需詳細資訊，請參閱本隱私權聲明的同步處理並備份設定一節。

    相機應用程式. 如果您允許相機應用程式使用您的位置，您使用裝置拍攝的相片和影片中將的內嵌位置資料。 其他描述性資料 (例如相機型號和相片或影片拍攝日期) 也將內嵌到相片及影片中。 如果您選擇共用相片或視訊，您選擇與之共用的人和服務將可以存取任何內嵌的資料。 啟用之後，您可以隨時透過關閉裝置 [設定] 功能表中對位置服務的存取，或關閉相機應用程式對位置服務的存取，來停用相機應用程式對您位置的存取。

    當 [相機] 應用程式開啟時，它會針對影像中可能用於影像增強功能的區域，顯示選取的相機所偵測到的矩形。 [相機] 應用程式不會保留任何影像增強資料。 您隨時都可以在 [相機] 應用程式的 [設定] 功能表或 [Windows 設定] 功能表中變更相機存取設定。

    [相片] 應用程式. [相片] 應用程式可協助您組織及共用相片。 例如，[相片] 應用程式會依據日期、位置、標記和面部，以不同的方式來將相片和影片分類。 [集錦] 索引標籤會依據拍攝日期來顯示相片和影片。 [相簿] 索引標籤可協助使用者依據位置和常用標記來組織他們的相片和影片。 [人員] 索引標籤可協助您整理相片和影片，只要將相片和影片與類似的面孔組成群組，您就可以將之與連絡人進行連接。

    如果您在 [相片] 應用程式的設定頁面上啟用「人員」設定，該應用程式就會使用面部分組技術將您的照片與影片分類整理。 該分組功能可以偵測相片或影片中的臉部，並判斷它們是否與本機相片集錦中其他相片和影片中的臉部類似。 您可以選擇使用 [人員] 應用程式將面部群組與連絡人相關聯。

    朗讀程式. 朗讀程式是一個協助您在沒有螢幕情況下使用 Windows 的螢幕閱讀應用程式。 當您遇到未描述的影像和不明確的連結時，朗讀程式會提供智慧型影像與頁面標題描述以及網頁摘要。

    您按下朗讀程式 + 按兩次 S 以選擇取得網頁的熱門連結清單時，系統會將您瀏覽中網站的 URL 傳送至 Microsoft，以產生熱門連結的摘要並提供和改善 Microsoft 服務 (例如前面＜Bing＞一節中所述的 Bing 服務)。

    您隨時都可以移至 Windows 設定應用程式中的 [朗讀程式] > [取得影像描述、頁面標題和熱門連結]，來停用這些功能。

    Windows Media Player#

    Windows Media Player 可讓您播放 CD、DVD 和其他數位內容 (例如 WMA 及 MP3 檔案)、擷取 CD，以及管理您的媒體櫃。 播放媒體櫃中的內容時若要加強您的體驗，Windows Media player 可顯示相關媒體資訊，例如專輯標題，歌曲標題、專輯封面、演出者及作曲者。 若要擴充媒體資訊，Windows Media player 會傳送要求給 Microsoft，其中包含標準電腦資訊、媒體內容識別碼，以及 Windows Media Player 媒體櫃中已包含的媒體資訊 (包括您已編輯或自行輸入的資訊)，如此 Microsoft 便可在辨識曲目後，傳回可用的額外資訊。

    Windows Media Player 也可讓您播放透過網路串流給您的內容。 若要提供這項服務，Windows Media Player 必須與串流媒體伺服器通訊。 這些伺服器通常由非 Microsoft 內容提供者操作。 播放串流媒體時，如果串流媒體伺服器提出要求，Windows Media Player 會傳送記錄檔到串流媒體伺服器或其他網頁伺服器。 記錄檔包含這類的詳細資料︰ 連線時間、IP 位址、作業系統版本、Windows Media Player 版本、Player 識別碼 (Player ID)、日期和通訊協定。 為保護您的隱私權，Windows Media Player 預設傳送播放機識別碼，隨工作階段的不同，識別碼也各有差異。

    Windows Hello#

    Windows Hello 提供透過生物特徵辨識驗證迅速存取裝置的方式。 如果啟用 Windows Hello，它會依據從相片中擷取並儲存在裝置上做為範本的一組獨特的點或特徵，來使用您的臉部、指紋或虹膜辨識您的身分 (但不會儲存您的臉部、指紋或虹膜的實際影像)。 當您登入時使用的生物特徵辨識驗證資料並不會離開您的裝置。 您的生物特徵辨識驗證資料會保留在裝置上，直到您將他移除。 不過，在 Windows Hello 閒置一段時間之後，系統會提示您以確認是否要繼續儲存生物特徵辨識驗證資料。 您可以在 [設定] 中刪除您的生物特徵辨識驗證資料。 深入了解 Windows Hello。

    Windows Search#

    Windows Search 可讓您從單一位置搜尋您的內容和網頁。 如果您選擇使用 Windows Search 搜尋「您的內容」，它將會在您的個人 OneDrive、您的商務用 OneDrive (若已啟用)、這些第三方提供者所支援的其他雲端儲存空間提供者，以及在您的裝置上提供項目的結果。 如果您選擇使用 Windows Search 來搜尋網路，或透過 Windows Search 取得搜尋建議，您的搜尋結果將由 Bing 提供，且我們將依據本隱私權聲明之 Bing 一節中的說明使用您的搜尋查詢。 深入了解 Windows 中的搜尋功能 .

    您的手機#

    [您的手機] 應用程式可讓您連結 Android 手機與 Windows 裝置，以獲得各種不同的跨裝置體驗。 您可以使用 [您的手機]，在 Windows 裝置上查看 Android 手機中最近的相片；在 Windows 裝置上透過 Android 手機撥打電話或接聽來電；從 Windows 裝置檢視和傳送簡訊；從 Windows 裝置檢視、關閉 Android 手機通知，或對這些通知執行其他動作；以及在 Windows 裝置上透過 [您的手機] 的鏡像功能分享手機畫面。

    若要使用「您的手機」，「您的手機」應用程式必須安裝在 Windows 裝置上，而 Android 手機上則必須安裝「您的手機小幫手」應用程式。 在 Windows 裝置上啟動「您的手機」應用程式時，您會收到提示訊息要求提供您的行動電話號碼。 我們只會使用這個行動電話號碼來傳送連結給您，提供下載「您的手機小幫手」應用程式的資訊。

    若要使用「您的手機」，您必須在 Windows 裝置上的「您的手機」應用程式以及在 Android 手機上的「您的手機小幫手」應用程式中登入 Microsoft 帳戶。 您的 Android 手機必須連線至 Wi-Fi，而您的 Windows 裝置則必須連線到網際網路，並允許「您的手機」在背景中執行。 若要使用「您的手機」的鏡像功能，Android 手機也必須啟用藍牙。 使用「您的手機」時也需要在 Windows 裝置上設定 Windows Hello，當做額外的安全性措施。

    Microsoft 在提供「您的手機」的各項功能給您時，會同時收集效能、使用方式以及裝置資料，其中包括，例如，您的行動電話和 Windows 裝置的硬體功能、「您的手機」工作階段的數目和持續時間，以及您在設定期間所花費的時間。

    您可以隨時在 accounts.microsoft.com/devices 使用 Microsoft 帳戶登入並更新 Android 手機上的設定，以取消 Android 手機和 Windows 裝置的連結。 如需詳細資訊，請參閱我們的支援頁面。

    簡訊. 「您的手機」可讓您在 Windows 裝置上檢視傳送到 Android 手機的簡訊，並從 Windows 裝置傳送簡訊。 只有在過去 30 天內所接收和傳送的簡訊會顯示在您的 Windows 裝置上。 這些簡訊是暫時儲存在您的 Windows 裝置上。 我們不會在我們的伺服器上儲存您的簡訊，也不會變更或刪除您 Android 手機上的任何簡訊。 您可以看到透過 SMS (簡短訊息服務) 和 MMS (多媒體訊息服務) 所傳送的簡訊，但無法看到透過 RCS (豐富通訊服務) 所傳送的訊息。 為了提供這項功能，「您的手機」會存取您的簡訊內容以及您接收或傳送簡訊的對象 (個人或企業) 的連絡資訊。

    通話. [您的手機] 可讓您在 Windows 裝置上透過 Android 手機撥打電話和接聽來電。 您還可以透過 [您的手機]，在 Windows 裝置上檢視最近的通話。 若要啟用此功能，您在 Windows 裝置和 Android 手機上都必須啟用特定權限 (例如通話記錄檔存取權，以及從電腦撥打電話的權限)。 您隨時都可以在 Windows 裝置的[您的手機設定] 頁面以及 Android 手機的設定中撤銷這些權限。 在 Windows 裝置上的通話記錄檔中，只能看到過去 30 天內所接收和撥打的通話。 這些通話詳細資料是暫時存放在 Windows 裝置上。 我們不會變更或刪除您的 Android 手機上的通訊記錄。

    相片. 「您的手機」可讓您在 Windows 裝置上複製、分享或編輯 Android 手機上的相片。 Android 手機上的 [手機相簿] 和 [螢幕擷取畫面] 資料夾中最近的相片並不會隨時全部都顯示在您的 Windows 裝置上。 這些相片是暫時儲存在您的 Windows 裝置上，當您在 Android 手機上拍攝更多相片時，我們就會從您的 Windows 裝置移除較舊的相片暫時複本。 我們不會在我們的伺服器上儲存您的相片，也不會變更或刪除您 Android 手機上的任何相片。

    通知. 「您的手機」可讓您在 Windows 裝置上檢視 Android 手機上的通知。 透過「您的手機」，您可以從 Windows 裝置讀取及關閉 Android 手機上的通知，或執行和通知有關的其他動作。 若要啟用「您的手機」的這項功能，您必須取得某些權限，例如可在 Windows 裝置和 Android 手機上同步通知的權限。 您可以隨時在 Windows 裝置上「您的手機」的 [設定] 頁面以及 Android 手機的設定中撤銷這些權限。 如需詳細資訊，請參閱我們的支援頁面。

    鏡像. 「您的手機」可讓您在 Windows 裝置上檢視 Android 手機的畫面。 Android 手機畫面會以像素串流的方式顯示在您的 Windows 裝置上，而當 Android 手機畫面透過「您的手機」連結至 Windows 裝置時，您在 Android 手機畫面上所啟用的任何音訊，則是透過 Android 手機來播放。

    文字轉換語音. 「您的手機」包含了協助工具功能，例如文字轉換語音。 您可以啟用文字轉換語音功能，它可讓您以音訊方式聽到簡訊或通知的內容。 如果啟用這項功能，在收到簡訊和通知時就會大聲讀出它們的內容。
